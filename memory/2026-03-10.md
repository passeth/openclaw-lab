## 크론 파이프라인 수정 (xAI Grok 검색 문제 해결)

### 문제 발견
- `web_search` 도구 with provider="grok" → 타임아웃/중단
- 직접 API 테스트 결과: `search_parameters` 방식 deprecated

### 해결
**xAI 새 API 구조:**
- 엔드포인트: `/v1/responses` (not `/v1/chat/completions`)
- 도구: `{"type": "web_search"}` (built-in, not function calling)
- 모델: `grok-4-1-fast-reasoning` (grok-4 패밀리만 서버사이드 도구 지원)
- 입력: `input` 배열 (not `messages`)

**테스트 성공:**
```bash
curl -X POST "https://api.x.ai/v1/responses" \
  -H "Authorization: Bearer $XAI_KEY" \
  -d '{"model":"grok-4-1-fast-reasoning","input":[...],"tools":[{"type":"web_search"}]}'
```
→ 9회 자동 검색 + 종합 응답 생성
→ Paula's Choice 크림: 가격 $37~42, 평점 4.4~4.5/5, 리뷰 요약
