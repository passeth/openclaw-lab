## 4. USP (Unique Selling Point)

### ★ CORE USP — 천연 이중 스크럽 시스템

```
[1차] Sea Salt + Microcrystalline Cellulose  →  딥 각질 + 부드러운 마무리
[2차] Cellulose Powder + Apricot Seed        →  세밀한 결 정돈 + 자극 최소화
```

**셀룰로오스 스크럽의 차별점 (마케팅 핵심):**
- **구형(球形) 입자** — 마이크로비즈와 달리 모서리가 없어 피부 미세 손상 없음
- **완전 생분해** — 자연에서 100% 분해 / 친환경 인증 소구 가능 (중국 Z세대 선호)
- **균일한 각질 제거** — 살구씨(불규칙 입자) 단독 대비 균일한 각질 제거력
- **식물성 원료** — 목재 펄프·면 유래 / 비건·클린 소구 가능

**이중 스크럽 시너지 포인트:**
> "Sea Salt의 강력한 미네랄 각질 케어 + Cellulose의 부드러운 마무리"
> → 각질은 확실히, 자극은 없게 — 기존 스크럽 제품의 딜레마 해결

---

### ① 이중 스크럽 × 이중 광택 (中 시장 핵심 클레임)

**각질 케어**: 天然이중스크럽(天然双重磨砂) — Sea Salt + 셀룰로오스
**피부 광택**: Niacinamide + Glutathione + Pearl Extract 미백 트리오
→ **"각질 제거 후 즉각 빛나는 피부" / 冷白皮 트렌드 직접 소구**
→ 팔꿈치·무릎·목·겨드랑이 부위 집중 케어 마케팅

### ② 히알루론산 + 식물 추출물 세정 중 보습

Sodium Hyaluronate + 꽃 추출물 복합(히비스커스·목련·카모마일·장미 등)
→ 스크럽 후 당기지 않는 촉촉한 피부결
→ "씻는 동안 보습" 컨셉 — 중국 소비자 선호 복합 기능 소구

### ③ 판테놀·알란토인 즉각 진정 (스크럽 후 케어)

스크럽 물리 자극 → 판테놀·알란토인 즉각 진정
→ 빨개짐·당김 없는 부드러운 사용감

### ④ 향 3종 선택 경험

Pure(차콜) / Soft(Givaudan White Musk V2) / Musk(Allergen Free White Musk)
→ 동일 처방 베이스, 취향 맞춤 선택 → 세트 구매 유도 가능
