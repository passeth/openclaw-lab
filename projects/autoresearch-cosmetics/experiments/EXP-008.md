### H1: Saccharomyces vs Lactobacillus 쌀 발효 — 마이크로바이옴 조절 비교

#### 핵심 발견

```
원료별 프로필:

Saccharomyces/Rice Ferment Filtrate (사케 발효 유형):
├─ 주요 생성물: 펩타이드, 아미노산, B 비타민, β-글루칸, 유기산, 항산화제
├─ 작용 기전: 두피 장벽 강화, 수분 유지, 항산화 방어, ATP 생산 지원
├─ 마이크로바이옴 효과: 간접적 (장벽 강화 → 환경 안정화 → 다양성 유지)
├─ 우려사항: "Malassezia 먹이" 성분 목록 일부 포함 — 단, 비활성화(deactivated) 여과물에서는
│   실제 위험 미입증
└─ 상업 우위: 상용화 범위 넓음 (Redken, Briogeo, Daeng Gi Meo Ri 등)

Lactobacillus/Rice Ferment Extract (유산균 발효 유형):
├─ 주요 생성물: 유기산(젖산), 박테리오신, 프리바이오틱 대사산물
├─ 작용 기전: pH 조절, 항균/항진균 활성, 마이크로바이옴 직접 조절
├─ 마이크로바이옴 효과: 직접적 (Malassezia 직접 억제, 유익균 성장 지원)
└─ 상업 현황: 두피 케어에서 희귀 (Whamisa 오일리 스캘프 샴푸 정도)

임상 근거 (2023):
└─ SLFC(Saccharomyces + Lactobacillus 복합체, n=~30, 중국 여성):
   ├─ 세균: α-다양성 증가, ↑ Cutibacterium acnes(유익), ↑ S. epidermidis
   ├─ 진균: β-다양성 변화, M. restricta 조절 (감소 방향)
   └─ 복합 사용 → 시너지 효과 (단일 원료 비교 임상 없음)

2025 in vitro (Lactobacillus 쌀 발효):
└─ Lactobacillus spp.가 Malassezia, Trichophyton 억제
   S. aureus 바이오필름 감소
   VEGF 상향 조절 (모발 성장 관련)
```

#### Malassezia 억제 우위 비교

| 항목 | Saccharomyces | Lactobacillus | 비고 |
|------|--------------|--------------|------|
| 직접 항진균 | 간접적 | 직접적 | 박테리오신/유기산 |
| 임상 데이터 | SLFC 혼합으로 존재 | in vitro만 | 2023/2025 |
| 유익균 다양성 | ↑ (SLFC 데이터) | ↑ (추정) | 복합체 기준 |
| 상업 검증 | 광범위 | 제한적 | — |
| 위험(Malassezia 먹이) | 이론적 위험, 미입증 | 없음 | — |

#### 판정: **부분지지**
> 직접 비교 임상은 없음. Lactobacillus가 Malassezia 직접 억제에서 우세(in vitro). Saccharomyces는 상업 검증이 넓고 SLFC 복합 임상에서 효과 확인. 최적 전략: **두 타입 복합** 또는 **현재 처방(Saccharomyces)에 Lactobacillus 추가 검토**
> 신뢰도: **3/5** (직접 비교 데이터 부재)

### H2: 아시아인 두피 마이크로바이옴 특이성 및 포스트바이오틱 반응성

#### 핵심 발견

```
아시아인 두피 baseline 조성 (2023-2025 데이터):

세균 조성 (건강한 두피):
├─ 중국 여성(n=63): Actinomycetota 65%, Bacillota 34%
│   Cutibacterium + Staphylococcus >95%
│   → 서양인과 주요 세균 속 동일, 비율 유사
├─ 싱가포르(n=65, 아시아 다수): 건강 두피 C. acnes ~77-84%, Malassezia ~7%, Staphylococcus ~3%
└─ 다민족 연구(n=212): White/Black에서 S. epidermidis 더 높음, 아시아인에서 상대적 낮음

진균 (Malassezia) 특이성 ⚠️ 중요:
├─ 아시아인 비듬: M. restricta 증가 없음 (서양인은 증가함) ← 핵심 차이
├─ 중국인: M. restricta 우세하나 비듬 시 S. capitis + M. restricta 병인
├─ 싱가포르 D/SD: Malassezia:C. acnes 비율 ~4x 상승
└─ 아시아인 비듬: M. globosa 감소 (전 민족 공통)

아시아인 비듬의 특이적 병인:
├─ 서양인: 주로 M. restricta 우세 (항진균 단독 효과적)
├─ 아시아인: 세균성 dysbiosis(S. aureus ↑, C. acnes ↓) + 진균 동시 관여
└─ → 항진균 + 항세균 + 균형 복원의 3중 전략 필요

포스트바이오틱 반응성 (아시아인 대상 연구):
├─ SLFC 임상(2023, 중국 여성): 두피 마이크로바이옴 조절 + 증상 완화 확인
│   이소부탄산/이소발레르산 감소(S. epidermidis 대사산물) → 피지 조절
├─ C. acnes 포스트바이오틱(2024, 싱가포르): 1% sodium propionate 로션
│   비듬 점수 2배 감소, Malassezia:C. acnes 비율 정상화
└─ 아시아인 전용 발효 성분 반응성 연구: 아직 부재 (서양 비교 데이터 없음)
```

#### 아시아인 두피 처방 시사점

```
K-Ferment Biome Shield 아시아인 최적화:

기존 처방 → 아시아인 특화:
├─ PO 0.5%: Malassezia 직접 억제 ✅ (M. restricta 표적)
├─ 쌀 발효 여과물 2%: 유익균 복원 ✅ (C. acnes/S. epidermidis ↑)
├─ Zinc PCA 1%: 피지 조절 + S. aureus 환경 억제 ✅
└─ 추가 검토: S. aureus 억제 활성 강화 필요

아시아인 특이 S. aureus 대응:
├─ 현재 처방: ZincPCA가 간접적 억제 효과
├─ 강화 옵션: Centella asiatica 추출물 또는 Tea Tree Oil (제한적)
└─ 포스트바이오틱: C. acnes 대사산물(propionate) 추가 — 실험적
```

#### 판정: **지지**
> 아시아인 두피는 서양인 대비 M. restricta 패턴 다름, 세균성 dysbiosis 비중 더 높음 확인. 포스트바이오틱(SLFC, C. acnes 포스트바이오틱) 아시아인 대상 임상에서 효과 확인됨. K-발효 쌀 특이적 반응성 비교 연구는 부재.
> 신뢰도: **3.5/5** (아시아인 특이 K-발효 반응 데이터 부재로 감점)
