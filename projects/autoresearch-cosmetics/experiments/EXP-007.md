#### 핵심 발견: **1-3% 범위 확립, 아미노산 계면활성제와 우수한 호환성**

```
최적 농도 근거:

├─ Bloomage Biotechnology (microHA™): 0.5-4% (모발 케어)
├─ CN115089535B (2022): 2-3% 최적, 총 active 1-5% 허용

└─ 권고: 2% = 황금 농도 (효능 + 비용 균형)

├─ 쌀 발효 여과물 pH: ~4-6 (자연 산성)

### H2-3: 4-Active + K-발효 완전 처방 안정성 및 비용 분석

#### 핵심 발견: **PO 0.5% + 발효 쌀 추출물 병용 — 우수한 안정성**

```
PO 안정성 프로필:

pH: pKa ~7.4 → pH 3-9 범위 안정
     SF 샴푸 목표 pH 5.0-5.5에서 최적 용해도/활성
색상: 투명/클리어 제형 가능 (ZPT 불투명 대비 우위)
     알데히드/케톤계 향료 과다 시 드물게 미세 변색 가능
향: PO 자체 무취에 가까움, 향료 안정성 문제 없음
가속: 45°C 3개월 = 2-3년 shelf life 상당
     1년 >40°C: 점도/pH 안정 (약간 농화는 이점)

4-Active 완전 조합 안정성:
├─ PO 0.5% + 쌀 발효 여과물 2%: Off & Relax SPA 시리즈 상용화 확인
├─ PO 0.5% + Zinc PCA 1%: 동일 제품 계열에서 확인
├─ PO + Rice Ferment + Zinc PCA 삼중 조합: 안정성 이슈 보고 없음
└─ Niacinamide 추가 시: pH 5.0-5.5에서 니아신아미드 전환 최소화 가능
```

```
처방 포지셔닝:
├─ 원가율 목표: 15-20% (소비자가 기준)
├─ 적정 소비자가: 47,250-94,700원/500ml

### H3-3: 규제 허용 범위 및 효능 시험 설계

#### 핵심 발견: **3개 시장 모두 일반 화장품으로 클레임 가능 — 단, 입증 필수**

```
한국 MFDS:
├─ 기능성 화장품 해당 여부: ❌ 해당 없음 (항비듬/마이크로바이옴)
│   → 탈모 완화는 기능성 해당하나, 비듬 케어/마이크로바이옴 균형은 일반 화장품
├─ 클레임 자유도: 높음 (허위/과대광고 금지 원칙)
├─ 입증 필요: 안전성/효능 데이터 (임상 시험, in vitro)
└─ 주의: "탈모 치료", "비듬 치료" 등 의학적 클레임 금지

├─ 클레임 기준 6가지: 합법성, 진실성, 증거, 정직성, 공정성, 정보적
├─ 마이크로바이옴 클레임: 신흥 영역 — 미생물총 "균형", "지지" 가능
├─ 항비듬 클레임: ZPT 2022 금지 → PO 기반으로 전환 활발
├─ 입증 수준: 임상 시험 선호 (최소 n=20, 피부과 평가)
└─ "치료", "의약" 언어: 금지

ASEAN (ACD Appendix III):
├─ 항비듬: 화장품-의약 경계선이지만 비치료적 언어로 허용
├─ 마이크로바이옴: 미언급 → 일반 원칙 적용
├─ 입증: in vitro + 소비자 인지 + 임상 (클레임 강도 비례)
└─ 사후 시장 감시 체계 (사전 승인 불필요)
```

#### 권장 효능 시험 설계

```
최소 입증 패키지 (3개 시장 공통):

1. In Vitro 마이크로바이옴 연구
   - Mock Dandruff Community (M. restricta + S. epidermidis + C. acnes)
   - 처리: 완제품 희석액 (1%, 5%) 24-72h
   - 측정: 균종별 대사 활성, 유익균:병원균 비율
   - 기대: Blend 162 방법론 준용 (WO2024118975A1)

2. 두피 장벽 in vitro
   - 3D 두피 오가노이드 또는 재건 표피
   - 처리: 30분 (린스오프 시뮬레이션)
   - 측정: TEWL, Filaggrin 발현, Ceramide 함량
   - 기대: PENTAVITIN® 데이터 수준 (Filaggrin ↑)

3. 소비자 임상 (최소 n=30, 4주)
   - 대상: 두피 비듬/가려움 있는 성인
   - 처리: 주 3회 사용
   - 측정: 비듬 점수 (피부과 평가), 소비자 설문
   - 선택: 16S rRNA 시퀀싱 (마이크로바이옴 변화, n=10 서브셋)

4. 안정성 시험 (신뢰도 보강)
   - 피부 1차 자극 (패치 테스트)
   - 반복 사용 자극
   - 두피 민감성 평가

비용 추정:
├─ In vitro 세트: 500-800만원
├─ 소비자 임상: 1,500-3,000만원
├─ 마이크로바이옴 시퀀싱: 300-500만원
└─ 총 최소 입증 패키지: 약 2,300-4,300만원
```

#### 판정: **지지** ✅
> 3개 시장 공통으로 "마이크로바이옴 균형 지원" 클레임 가능. 올바른 언어 선택과 최소 입증 패키지(in vitro + 소비자 임상)로 2,300-4,300만원 투자로 글로벌 클레임 구축 가능.
