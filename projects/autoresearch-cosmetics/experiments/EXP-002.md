# EXP-002: SF 샴푸 세번 컨트롤 액티브 최적 조합 — 다중 타겟 전략

> 연구일: 2026-03-10  
> 사이클: 1 (탐색)  
> 상위 프로젝트: 두피 세번 컨트롤 SF 샴푸  
> 이전 실험: EXP-001 (Zinc PCA + 아미노산 계면활성제 안정성)

---

## 가설

> **H1:** Piroctone Olamine + Zinc PCA 병용이 단독보다 세번 감소율이 높다  
> **H2:** Niacinamide 1~2%가 두피 환경에서 피지 조절에 유의미한 효과를 보인다  
> **H3:** SA 없이도 PHA(Gluconolactone)가 두피 각질 관리에 효과적이다

---

## 탐색 결과

### H1: Piroctone Olamine + Zinc PCA 병용 시너지

#### 핵심 임상 근거

| 연구 | 피험자 | 처치 | 세번/유분 결과 |
|------|--------|------|---------------|
| **Ge et al. (2025)** J Cosmet Dermatol 24(1):e16742 | 중등~중증 두피 SD 20명 | SA + PO + Zinc PCA 프리겔/클렌저 4+12주 | 유분 스코어 2.60→1.40(4주, ↓46%) → 1.00(16주, **↓62%**, p<0.01) |
| **Sharifi-Rad et al. (2015)** | 비듬 30명 | 1% PO + Zinc PCA 허벌 샴푸 8주 | 100% 비듬 해소(5주), 두피 균형 개선(유분 정량 데이터 없음) |
| **Rakizadeh et al.** (학회발표) | 저항성 비듬 30명 | 1% PO + 1% Zinc PCA + 허벌 4주 | 세번 분비↓, 비듬/가려움 억제(정량 데이터 없음) |

#### 시너지 메커니즘

```
┌─ Zinc PCA ──────────────────────────┐   ┌─ Piroctone Olamine ──────────┐
│ • 5α-reductase 억제 → DHT↓          │   │ • Malassezia 세포막 파괴      │
│ • 피지선 활성 직접 조절              │   │ • 항진균 → 지루 악순환 차단    │
│ • 항균(P. acnes MIC 0.1-0.25%)      │   │ • 두피 염증 간접 감소          │
│ • PCA = NMF → 두피 보습 유지        │   │                               │
└──────────────┬──────────────────────┘   └──────────────┬────────────────┘
               │                                         │
               └──────────── SYNERGY ────────────────────┘
                  세번 과잉 → Malassezia 증식 → 염증 → 세번↑
                  이 악순환을 양쪽에서 동시 차단
```

#### 판정: **부분 지지** ⚠️
- 병용 처방의 세번 감소 효과는 임상적으로 확인(↓62%)
- 그러나 **단독 vs 병용 직접 비교 RCT는 존재하지 않음**
- Ge et al. 연구에 SA가 포함되어 있어 PO+ZnPCA 듀오만의 기여분 분리 불가

### H2: Niacinamide의 두피 피지 조절

#### 핵심 근거

| 연구 | 대상 | 농도 | 결과 |
|------|------|------|------|
| **Draelos et al. (2006)** PMID:16766489 | 일본 여성 100명 (안면) | 2% 모이스처라이저 4주 | SER **↓21.8%** vs 위약 10.7% (p=0.027) |
| Draelos et al. (2006) | 백인 여성 30명 (split-face) | 2% 6주 | CSL 감소 근유의(p=0.055), 광택↓(p=0.009), 유분↓(p=0.0002) |
| Fabbrocini et al. | 지루성 피부염 | 4% 토피컬 | 총 증상 스코어 **↓75%** |

#### 피지 억제 메커니즘

```
Niacinamide → 피지세포(Sebocyte)
  ├─ SCD1 (Stearoyl-CoA Desaturase-1) 하향조절
  ├─ ACSL1 (Acyl-CoA Synthetase) 하향조절
  ├─ → 지질 신생합성(De novo lipogenesis) ↓25-35%
  ├─ 모낭관 각질 제거 → 피지 저장소 감소
  └─ 항염(사이토카인↓) + 세라마이드↑ → 장벽 강화
```

#### 두피 적용 시 고려사항
- **모든 임상 데이터가 안면 피부 기반** — 두피 전용 RCT 없음
- 안면과 두피의 피지선 생물학은 유사하므로 **외삽 가능성은 높음**
- 샴푸(린스오프)에서의 유효 농도: **2%** (안면 데이터 기준)
- 리브온(두피 세럼) 형태에서 더 효과적일 가능성 (접촉 시간↑)

#### 판정: **부분 지지** ⚠️
- 안면 피지 억제 메커니즘 및 효과는 견고하게 입증
- 두피 직접 근거 부재가 주요 한계
- 1% 농도에서의 데이터 없음 (최소 유효 농도는 2%)

### H3: PHA(Gluconolactone) — SA 대체 각질 제거

#### 핵심 근거

| 연구 | 대상 | 처치 | 결과 |
|------|------|------|------|
| **2024 Open-label (n=29)** Cureus | 경~중등도 SD/비듬 | AHA+BHA+PHA 스크럽 15일 | ASFS **↓61.24%** (p<0.0001), 보습↑76%, 각질 65.5%→6.9% |
| 피부 연구 다수 | 반응성 피부 | Gluconolactone | 무자극, 로사시아/습진에도 안전 |

#### PHA vs BHA vs AHA 비교

| 특성 | PHA (Gluconolactone) | BHA (Salicylic Acid) | AHA (Glycolic Acid) |
|------|---------------------|---------------------|---------------------|
| 분자 크기 | **대형** (침투↓) | 중형 (모공 침투) | 소형 (표면 침투) |
| 지용성 | ✗ | ✓ (피지 관통) | ✗ |
| 각질 제거력 | 중간 (완만) | **강함** (빠름) | 강함 |
| 자극도 | **최저** | 중~고 | 중간 |
| 보습 | ✓ (휴멕턴트) | ✗ (건조) | △ |
| 항산화 | ✓ | ✗ | ✗ |
| 모공 정화 | △ (표면) | **✓** (깊이) | △ |
| 민감 두피 | **최적** | 주의 필요 | 주의 필요 |
| NMPA 규제 | **제한 없음** ✅ | 리스크(약품 분류 가능) | 제한 있음 |

#### 판정: **지지** ✅
- SA 대비 각질 제거 효과는 온화하지만 유효
- 민감 두피 적합성 우수
- NMPA 규제 리스크 없음 — 중국 시장 적합
- **다만** SA의 모공 깊이 침투 + 항염 효과를 100% 대체하지는 못함
- 권장 사용: 5-10% 리브온 또는 2-5% 린스오프

## 실무 권장사항

### 즉시 적용 가능
1. **Gluconolactone 3%** 추가 — SA 대체 각질 제거제로 처방에 반영
2. **Piroctone Olamine 0.5% + Zinc PCA 0.3%** 병용 유지 (EXP-001 안정화 전략 적용)

### 추가 검증 필요
3. **Niacinamide 2%** — 두피 린스오프 조건에서의 유효성 확인 필요
   - 접촉 시간 3-5분의 린스오프 샴푸에서 2%가 충분한가?
   - Zinc PCA와의 화학적 호환성 검토 필요

### 안정성 주의
4. Niacinamide + Zinc PCA 동시 배합 시 pH 조건 확인 (Niacinamide는 pH 5-7 안정)
5. Gluconolactone 산성(pH 3.5-4.0) → 최종 pH 밸런스 재조정 필요
