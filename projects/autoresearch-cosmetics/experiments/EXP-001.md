# EXP-001: Zinc PCA + 아미노산 계면활성제 시스템 안정성 및 효능

> 연구일: 2026-03-10  
> 사이클: 1  
> 상위 프로젝트: 두피 세범 컨트롤 SF 샴푸

---

## 가설

> Zinc PCA(0.3%)는 아미노산계 음이온 계면활성제(Sodium Cocoyl Glycinate) 환경에서  
> pH 5.0~5.5 + 킬레이트제(EDTA-2Na) 조건 하에 안정적으로 배합 가능하며,  
> Piroctone Olamine과의 병용 시 세범 컨트롤 시너지를 발휘한다.

### 2. Zinc PCA + 음이온 계면활성제 상호작용

#### 핵심 문제: Zn²⁺ 이온과 음이온 계면활성제의 불용성 염 생성

| 메커니즘 | 설명 |
|----------|------|
| 해리 | Zinc PCA → Zn²⁺ + 2 PCA⁻ (수용액에서 부분 해리) |
| 반응 | Zn²⁺ + 2 R-COO⁻ (anionic surfactant) → Zn(RCOO)₂↓ (침전) |
| 영향인자 | pH, 온도, 계면활성제 종류, Zn²⁺ 농도, 이온 강도 |

#### 계면활성제별 상호작용 리스크

| 계면활성제 | 유형 | Zn²⁺ 결합력 | 침전 리스크 |
|-----------|------|------------|-----------|
| SLS/SLES | 황산염(sulfate) | 중간 | 중간 — SO₄²⁻ 기반, Zn과 비교적 용해성 |
| Sodium Cocoyl Glycinate | 아미노산(amino acid) | **높음** | **높음** — 카복실기(-COO⁻)가 Zn²⁺과 강한 킬레이트 |
| SLMI (Isethionate) | 이세티오네이트 | 낮음 | 낮음 — 술폰산 기반, Zn과 상호작용 약함 |
| Cocamidopropyl Betaine | 양쪽성 | 최소 | 최소 — 내부 전하 상쇄 |

**핵심 발견:**  
Sodium Cocoyl Glycinate의 카복실기(-COO⁻)는 Zn²⁺과 **킬레이트 결합**을 형성하는 경향이 강하다.  
이는 SLS 시스템보다 **오히려 더 높은 침전 리스크**를 의미한다.

→ 아미노산계를 선택한 것이 마일드 측면에서는 장점이나,  
   Zinc PCA 안정성 측면에서는 **추가 대응이 반드시 필요**한 trade-off

### 3. 안정화 전략 평가

| 전략 | 메커니즘 | 효과 | 실용성 |
|------|----------|------|--------|
| **EDTA-2Na (0.05~0.1%)** | Zn²⁺을 EDTA가 먼저 킬레이트 → 계면활성제와의 결합 차단 | ⚠️ 양날의 검 | ★★★ |
| **pH 최적화 (5.0~5.5)** | pH↓ → Zinc PCA 해리도↓ → 유리 Zn²⁺ 감소 | ★★★★ | ★★★★★ |
| **투입 순서 제어** | Zinc PCA를 계면활성제 미셀 형성 후 첨가 → 유리 Zn²⁺ 최소화 | ★★★ | ★★★★ |
| **Phytic Acid (0.05%)** | 천연 킬레이트, EDTA 대체 가능 | ★★★ | ★★★ |
| **Zinc PCA 농도 최적화** | 0.3% → 필요시 0.2%로 하향 | ★★★★ | ★★★★★ |
| **SLMI 비율 증가** | 이세티오네이트 비율↑ → Glycinate 비율↓ → Zn 결합 사이트↓ | ★★★★ | ★★★★ |

#### EDTA 양날의 검 문제

```
EDTA + Zn²⁺ → [EDTA-Zn]²⁻ (안정한 킬레이트)
```

- **장점:** Zn²⁺이 계면활성제와 반응하는 것을 방지
- **단점:** EDTA가 Zn²⁺을 너무 강하게 잡으면 Zinc PCA의 **피부 전달 효율이 떨어질 수 있음**
- **해결:** EDTA 농도를 0.05% 이하로 최소화 → Zn²⁺ 전부가 아닌 일부만 킬레이트
- **대안:** Phytic Acid — 킬레이트 강도가 EDTA보다 약해서 균형점이 더 좋을 수 있음

### 4. 임상 근거: SA + Piroctone Olamine + Zinc PCA 병용

**논문:** Ge L et al. "A Cohort Clinical Study on the Efficacy of Topical Salicylic Acid/Piroctone Olamine Dandruff Pre-Gel and Cleanser in Improving Symptoms of Moderate to Severe Seborrheic Dermatitis of the Scalp"  
*J Cosmet Dermatol.* 2025 Jan;24(1):e16742. PMID: 39778065

| 항목 | 결과 |
|------|------|
| 피험자 | 중등~중증 두피 지루성 피부염 20명 |
| 처치 | SA + Piroctone Olamine + Zinc PCA 프리겔 + 클렌저 |
| 기간 | 4주 집중 + 12주 유지 |
| 비듬 스코어 | 2.45 → 1.10 (p < 0.01) ✅ |
| 가려움 | 2.35 → 1.10 (p < 0.01) ✅ |
| 홍반 | 1.55 → 1.10 (p < 0.05) ✅ |
| **유분** | **2.60 → 1.40 (p < 0.01)** ✅ |
| 전체 개선율 | **80%** |

**시사점:**
- Piroctone Olamine + Zinc PCA 병용의 세범 컨트롤 효과가 **임상적으로 입증**됨
- 이 논문에서는 SA를 포함했으나, 우리 처방에서는 NMPA 리스크로 SA를 제외
- SA 없이도 Piroctone + Zinc PCA 듀오만으로 유분 조절 효과 기대 가능
- 다만 SA 제외 시 각질 제거 기능은 약화 → 보완 성분 검토 필요

### 5. 수정 처방 제안

기존 처방의 안정성 리스크를 반영한 수정안:

| 변경 항목 | 기존 | 수정안 | 근거 |
|-----------|------|--------|------|
| Sodium Cocoyl Glycinate | 10% (순분) | **8%** | Zn²⁺ 결합 사이트 감소 |
| SLMI | 4% (순분) | **6%** | Glycinate 감소분 보완 + Zn 비반응성 |
| EDTA-2Na | 0.05% | **Phytic Acid 0.05%** | 천연 킬레이트, Zn 전달효율 보존 |
| Zinc PCA 투입 순서 | 미지정 | **Phase B (계면활성제 완전 용해 후)** | 미셀 형성 후 투입 → 유리 Zn²⁺↓ |
| pH 관리 | 5.0~5.5 | **5.0~5.3 (하한 관리)** | Zinc PCA 해리 최소화 |

## 결론

1. **Zinc PCA + 아미노산 계면활성제 조합은 가능하지만, SLS/SLES보다 안정성 리스크가 높다**
   - 카복실기(-COO⁻)와 Zn²⁺의 킬레이트 경향이 핵심 원인

2. **다중 안정화 전략 병행 필요:**
   - Glycinate 비율↓ + SLMI 비율↑
   - 천연 킬레이트(Phytic Acid) 사용
   - 투입 순서 제어 + pH 하한 관리(5.0~5.3)

3. **Piroctone Olamine + Zinc PCA 병용 세범 컨트롤 효과는 임상 근거 확보**
   - SA 제외 시에도 유분 조절 기대 가능

4. **필수 후속:** 45°C 4주 가속 안정성 시험으로 실제 검증 필요
