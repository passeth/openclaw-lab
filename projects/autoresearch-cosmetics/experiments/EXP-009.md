│   ├─ Lactobacillus: 직접 항균/항진균 + pH 조절 + 유익균 지원
│   └─ 이론적 상호보완 ✅ 임상 증명 ❌

### H3: 쌀 발효 여과물 — Malassezia 먹이 우려 과학적 실체

#### 핵심 발견

```

├─ Dermazen 리스트: rice ferment filtrate를 "Malassezia-feeding" 분류
│   → 근거: 잔존 당류/발효산물이 Malassezia 성장 기질 가능성

└─ Sezia.co: rice ferment filtrate 고위험 분류

과학적 검토 (2022-2025):
├─ Malassezia 실제 영양 요구:
│   ├─ 필수: C11-C24 지방산 (불포화 > 포화), 특히 올레산(C18:1)
│   ├─ 탄수화물/당류: 주요 에너지원이 아님 — 외부 지방산 의존 필수
│   └─ → 당류 자체는 주요 성장 기질 아님 ← 핵심
├─ 쌀 발효 여과물 조성:
│   ├─ 주요 성분: 아미노산, 펩타이드, B 비타민, 유기산, β-글루칸
│   ├─ 지방 함량: 여과/스트레이닝 과정에서 제거 → 저지방
│   └─ 올레산: 쌀 기름(35-40%)이나 여과물에서 미미 — 미검증
├─ PubMed 근거 현황:
│   ├─ 쌀 발효 여과물의 Malassezia 성장 촉진 in vitro: 없음
│   ├─ 쌀 발효 여과물의 Malassezia 억제 in vitro: 없음
│   └─ → 우려는 커뮤니티 기반, 실험적 근거 없음
├─ 린스오프 제형 맥락 (샴푸):
│   ├─ 두피 접촉 시간: 30초-3분 (샴푸 사용 기준)
│   ├─ 실제 생물학적 유효 농도: 제형 희석 + 헹굼으로 대폭 감소
│   └─ → 이론적 위험이 제형 현실에서 유효할 가능성 낮음
└─ 반증:
   ├─ 쌀 발효 샴푸 브랜드(Luseta, Hiromi, Viori): 비듬 감소 효과 표방
   │   (임상 데이터 아님, 마케팅 주장)
   └─ Lactobacillus 쌀 발효(2025 IJSAT): Malassezia/Trichophyton 억제 in vitro
      → 쌀 발효 여과물이 오히려 항진균 활성 가능성
```

#### 위험 등급 재평가

| 측면 | 평가 | 근거 |
|------|------|------|
| 당류 = Malassezia 먹이 | ❌ 잘못된 단순화 | Malassezia는 지방산 의존, 당 아님 |
| 잔존 지방산 = 먹이 | ⚠️ 이론적 가능 | 여과물에서 지방 제거됨, 미검증 |

| 일반 두피 케어에서 | 안전 | PubMed 촉진 근거 없음 |

#### 판정: **가설 수정 (우려는 과학적 근거 약함)**
> Malassezia 먹이 우려는 커뮤니티 기원이며 실험적 근거 없음. 쌀 발효 여과물의 지방 함량은 여과 과정에서 제거되어 주요 Malassezia 기질(지방산) 공급 가능성 낮음. 샴푸 제형(린스오프)에서 실질 위험은 이론적 수준. 단, 활성 지루성 피부염 소비자에게는 패치테스트 권고 유지.
> 신뢰도: **4/5** (Malassezia 영양학적 근거 명확, 쌀 발효 특이 in vitro 부재로 감점)
