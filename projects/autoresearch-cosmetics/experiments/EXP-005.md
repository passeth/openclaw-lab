## 탐색 결과

### H1: 쌀 발효 여과물 — 두피 상재균 다양성

#### 직접 근거 (두피 마이크로바이옴)

| 연구 유형 | 대상 | 핵심 발견 | 한계 | 출처 |
|-----------|------|----------|------|------|
| In vitro | FRW 유래 프로바이오틱스 | *T. rubrum* 14-16mm 억제원, 단백질 변성 58% 억제 (항염) | Malassezia 미시험, 마이크로바이옴 프로파일링 없음 | IJSAT 2025 |
| 리뷰 (2025) | Rice Fermentation Products 전반 | *S. epidermidis* 성장 촉진 + *S. aureus*/*E. coli* 억제 → 상재균 우세 유도 | 두피 특이적 임상 없음 | Cosmetics 2025;12(1):29 |
| 학회 초록 | Sake 발효 성분 | "마이크로바이옴 및 대사체 조절을 통한 항노화" — *M. luteus* 매개 시사 | 상세 미공개 | SCCJ 2025 Fan et al. |

#### 간접 근거 (피부 마이크로바이옴 → 두피 외삽)

```
쌀 발효 여과물의 마이크로바이옴 조절 메커니즘:

1. pH 조절 (pH 4-5)
   └─ 젖산 0.54-1.44% → 산성 환경 조성
   └─ 산 내성 상재균(S. epidermidis) 유리, 병원균(Malassezia) 불리

2. 선택적 항균
   └─ S. epidermidis ↑ (성장 촉진)
   └─ S. aureus ↓ (억제)
   └─ E. coli ↓ (억제)
   └─ T. rubrum ↓ (14-16mm 억제원)

3. Postbiotic 산물
   └─ SCFA (단쇄지방산) → 병원균 생물막 파괴
   └─ 박테리오신 → 선택적 항균
   └─ 폴리페놀/폴리사카라이드 → 항산화 + 장벽 지원

4. 프리바이오틱 기능
   └─ 이눌린 강화 FRW → 생균수 2.3×10⁶ CFU/mL
   └─ 유익균 선택적 영양원
```

#### 핵심 갭

- **두피 16S rRNA 시퀀싱 기반 다양성(α/β diversity) 측정 임상: 전무**
- 대부분 피부(팔뚝/얼굴) 데이터 → 두피 외삽에 한계
- Aspergillus/Saccharomyces 발효 여과물 vs 자연발효(Lactobacillus) 구분 불명확

#### 판정: **부분 지지** ⚠️
> 메커니즘적 근거(선택적 항균, pH 조절)는 설득력 있으나, 두피 특이적 임상 데이터 부재

### H2: 발효 추출물 + Piroctone Olamine 병용 효과

#### 직접 근거

**PO+발효 추출물 직접 비교 연구: 없음** ❌

#### 간접 근거

| 데이터 | 내용 | 시사점 |
|--------|------|--------|
| FERMENZA® (석류-비트 발효 추출물) | *M. furfur* 93% 억제, MIC90 20μg/mL, 28mm 억제원 | 발효 추출물 자체가 강력한 항진균 가능 |
| Salix purpurea bark 발효물 | 비듬/두피 홍반 감소 임상 | 발효물의 독립적 항진균 효과 |
| Blend 162 샴푸 (pre/postbiotic) | PO보다 우수한 beneficial:non-beneficial 비율 개선 | Postbiotic이 PO를 대체/보완 가능 |

#### 잠재적 간섭 리스크 ⚠️

```
쌀 발효 여과물의 Malassezia 영양원 리스크:

  "Malassezia-feeding ingredients" 목록에 rice ferment filtrate 포함
  └─ 잔류 당류/탄수화물 → Malassezia 성장/생물막 촉진 가능
  └─ 시판 "rice water" 제품은 활성 농도 <0.1%로 영향 미미할 수 있음
  
  대응 전략:
  ├─ 고도 정제(발효 후 여과/정제)로 잔류당 최소화
  ├─ PO 0.5%가 Malassezia 억제 기본선 유지
  └─ 발효 조건 최적화(완전 발효 → 잔류당 ↓)
```

#### 보완적 메커니즘 분석

```
PO의 한계 + 발효 추출물의 보완:

PO (Piroctone Olamine 0.5%)
├─ 장점: Malassezia 광범위 살균
├─ 한계: 유익균도 함께 감소 → dysbiosis 리스크
└─ 결과: 리바운드 가능, 장기 사용 시 내성 우려

발효 추출물 (Postbiotic)
├─ 장점: 선택적 억제 (Malassezia ↓, S. epidermidis ↑)
├─ 장점: 장벽 강화 → 병원균 재증식 방지
└─ 한계: 살균력은 PO 대비 약함

병용 시나리오:
├─ PO: 급성 Malassezia 부하 감소 (1차 방어)
├─ 발효 추출물: 유익균 회복 + 장벽 강화 (2차 방어)
└─ 결과: 살균 + 생태계 복원 = 지속 가능한 두피 건강
```

#### 판정: **불확실** ❓
> 직접 시너지 데이터 없음. 메커니즘적으로 보완 가능성 있으나, Malassezia 영양원 리스크도 존재. 정제도에 따라 결과 상반될 수 있음.

### H3: 발효 성분 Postbiotic의 두피 장벽 강화

#### 직접 근거 (임상 데이터)

| 성분 | 시험 | TEWL 변화 | 기타 | 출처 |
|------|------|----------|------|------|
| Rice Power™ No.11a (A. oryzae + yeast + LAB) | RCT, N=15, 4주 | **-57%** (vs 위약) | SC 수분 +10.1mS (p<0.01) | JID 2023 |
| 발효 미강 추출물 (경구) | 동물, UVB | TEWL 상승 3.3배→1.2배 억제 | β-glucosidase 활성 회복 | Semantic Scholar |
| Galactomyces Ferment Filtrate | 임상 (피부) | TEWL 유의미 감소 | Tight junction ↑ | 복수 |

#### 세라마이드 생성

```
발효 쌀 성분의 세라마이드 경로:

1. Rice Power™ No.11a
   └─ 세라마이드 합성 유전자 상향조절
   └─ 3D 인체 표피 모델: 세라마이드 1.3배 ↑ (p<0.05)
   └─ 임상 N=13, 4주: 유의미한 세라마이드 증가

2. Sake Kasu (술지게미)
   └─ 양조 과정에서 스핑고지질 → 글루코실세라마이드 가수분해
   └─ 유리 세라마이드 직접 공급
   └─ 경구/외용 모두 장벽 지질 보충

3. 발효 미강 (Yeast + LAB 공발효)
   └─ 높은 세라마이드 함량
   └─ β-glucosidase 활성 회복 → 글루코실세라마이드→세라마이드 전환 촉진
   └─ UVB 손상 후 장벽 복구 가속
```

#### 항균 펩타이드 유도

| 펩타이드/인자 | 유도 성분 | 메커니즘 |
|--------------|----------|----------|
| SLPI | GFF (Galactomyces) | NRF2 활성화 → 항바이러스/항균 |
| β-defensin | 발효 추출물 전반 | 선천면역 강화 |
| Cathelicidin (LL-37) | 발효 추출물 전반 | 항균 + 면역조절 |

#### 장벽 강화 유전자/경로

```
발효 쌀 추출물의 장벽 관련 분자 타깃:

유전자 상향조절:
├─ FLG (Filaggrin) ↑ 1.2x → NMF 생성
├─ BLMH ↑ → Filaggrin → 아미노산 전환
├─ Loricrin ↑ → 각질층 구조 강화
├─ Claudin-1/4 ↑ → Tight junction 강화
├─ Caspase-14 ↑ → Filaggrin 처리
└─ Occludin, ZO-1 ↑ → 경표피 저항 증가

신호전달 경로:
├─ AHR 활성화 → 분화 + NMF 생성
├─ NRF2 활성화 → 항산화 (GPX2, NQO1) + 항균 펩타이드
└─ IL-37 ↑ / IL-33·CXCL14 ↓ → 항염

기타:
├─ β-glucan (sake kasu) → 피부 맨틀 무결성
├─ 폴리사카라이드 → 보습 + 장벽 지지
└─ ROS 소거 85-92% → 산화 스트레스 방어
```

#### 판정: **지지** ✅
> 임상(TEWL -57%, 세라마이드 1.3x), in vitro(유전자 발현), 동물 데이터 모두 일관. 피부 데이터이나 두피 외삽 합리적.
