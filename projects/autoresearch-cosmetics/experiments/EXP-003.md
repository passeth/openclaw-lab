# EXP-003: 세번 컨트롤 4-Active 조합 심화 — 전달·호환성·경로 분석

> 연구일: 2026-03-10  
> 사이클: 2 (심화)  
> 상위 프로젝트: 두피 세번 컨트롤 SF 샴푸  
> 이전 실험: EXP-002 (4-Active 멀티타겟 전략 탐색)

---

## 가설

> **H1:** 샴푸 린스오프(3-5분 접촉) 조건에서 Niacinamide 2%의 피지세포 도달량은 리브온 대비 현저히 낮아 유효성이 제한된다  
> **H2:** Zinc PCA + Niacinamide + Gluconolactone + Piroctone Olamine 4종 동시 배합은 pH 5.0-5.5에서 화학적으로 안정하다  
> **H3:** Zinc PCA(5α-reductase 경로)와 Niacinamide(지질합성 경로)는 독립적 상가 효과를 보인다

---

## 탐색 결과

### H1: Niacinamide 린스오프 유효성 — 전달 동역학

#### 침투 데이터

| 조건 | 추정 전달량 | 근거 |
|------|------------|------|
| **리브온 세럼 (2%, 수시간)** | SC+표피 10-30% | Draelos 2006; in vivo confocal Raman |
| **린스오프 샴푸 (2%, 3-5분)** | **표면 잔류 1-10%** | 계면활성제 세정력으로 >90% 제거 |
| **두피 모낭 경유 (shunt route)** | 리브온 대비 2-5x 증가 가능 | 두피 모낭 밀도 ~400/cm²; 카페인 샴푸 연구 참조 |

#### 핵심 분석

```
Niacinamide (MW 122 Da, 수용성)
├─ SC 투과: 수초 내 시작, 1시간 후 ~20-30μm 깊이
├─ 린스오프 3-5분: 초기 SC 흡착만 발생, 대부분 세정 시 제거
├─ 모낭 shunt route: 두피의 높은 모낭 밀도가 보상
│   → 카페인 샴푸 연구에서 린스오프에서도 모낭 침착 2-5x 증가 확인
├─ pH 영향: pH 5.0 → pH 7.4 대비 투과율 저하 (이온화↓)
└─ 결론: 린스오프에서 유효량 도달 가능하나 리브온 대비 크게 열위
```

#### 실무 시사점
- **린스오프 샴푸에서 Niacinamide 2%의 효과는 기대보다 제한적**
- 모낭 shunt route가 두피에서의 보완 경로이나, 정량적 근거 부족
- **대안 전략:**
  - Niacinamide 농도 상향 (4-5%) — 비용↑ 대비 효과 불확실
  - 2-step 시스템: 샴푸(세정) + 두피 세럼(Niacinamide 리브온) 권장
  - 샴푸 단독이면 Zinc PCA 중심으로 가고, Niacinamide는 보조적 역할

#### 판정: **지지** ✅ (가설이 맞음 — 린스오프에서 전달량 제한)

### H2: 4-Active 동시 배합 pH/안정성

#### pH 안정성 매트릭스

| 성분 | 최적 pH | 안정 범위 | pH 5.0-5.5 호환 |
|------|---------|----------|----------------|
| Zinc PCA | 5.0-6.0 | 4.0-6.0 | ✅ 최적 범위 내 |
| Niacinamide | 5.0-7.0 | 4.0-8.0 | ✅ 안정 |
| Gluconolactone | 3.4-4.5 (각질제거) | 안정 ~5.5 | ⚠️ 각질제거 효력 약간 감소, 보습/항산화는 유지 |
| Piroctone Olamine | 4.0-7.0 | 3.5-8.0 | ✅ 안정 |

#### 화학적 상호작용

| 조합 | 반응 리스크 | 상세 |
|------|-----------|------|
| ZnPCA + Niacinamide | **없음** ✅ | Niacinamide는 아미드 — Zn²⁺과 반응성 없음 |
| ZnPCA + Gluconolactone | **낮음** ✅ | Gluconolactone 서서히 가수분해 → 글루콘산 → 가용성 Zinc Gluconate 형성 (오히려 유익) |
| ZnPCA + PO | **없음** ✅ | 알려진 상호작용 없음 |
| Niacinamide + Gluconolactone | **없음** ✅ | 시판 제품에서 병용 사례 다수 (클렌저, 세럼) |
| Gluconolactone + PO | **없음** ✅ | 별도 경로, 상호작용 보고 없음 |

#### 침전 리스크 평가

```
pH 5.0-5.5에서:
├─ Zinc PCA 수용해도: >50% → 0.3% 배합에 문제 없음
├─ 전해질 부하: 아미노산 계면활성제의 -COO⁻가 주요 리스크 (EXP-001)
├─ Gluconolactone 가수분해: 느림 → 급성 pH 변동 없음
├─ 온도 영향: 45°C 이상에서만 리스크 증가
└─ 결론: pH 5.0-5.5 유지 시 4종 동시 배합 안정
    (단, EXP-001의 Glycinate 비율 조정 + 투입 순서 제어는 필수)
```

#### 주요 발견: Gluconolactone의 숨겨진 이점
- Gluconolactone → 글루콘산으로 서서히 가수분해
- 글루콘산은 Zn²⁺과 **가용성 킬레이트(Zinc Gluconate)** 형성
- Zinc Gluconate는 화장품에서 널리 사용되는 안전한 아연 형태
- → Gluconolactone이 **Zinc PCA 안정화에 오히려 기여할 가능성**

#### 판정: **지지** ✅

### H3: Zinc PCA + Niacinamide 경로 독립성

#### 이중 경로 모델

```
                    테스토스테론
                         │
          ┌──── Zinc PCA ─┤ (5α-reductase 억제)
          │              ↓
          │         DHT ↓↓↓
          │              │
          │         피지선 자극 ↓↓
          │              │
          │         피지 분비 ↓↓
          │              │
          │              ▼
ADDITIVE  │         총 세번 감소
EFFECT    │              ▲
          │              │
          │         지질 합성 ↓↓
          │              │
          │         TG + FFA ↓↓
          │              ↑
          └── Niacinamide ─┤ (SCD1/ACSL1 하향조절)
                           │
                      IGF-1 경로 간섭
```

#### 근거 요약

| 항목 | 내용 |
|------|------|
| 경로 독립성 | **확인** — 5α-reductase(호르몬 상류) vs SCD1/ACSL1(지질합성 하류) |
| 시너지 근거 | **없음** — 교차 강화(crosstalk/potentiation) 보고 없음 |
| 상가 효과 근거 | **간접 지지** — 2006 리뷰에서 병용 시 "보완적 항염/세번 억제" 언급 |
| 직접 비교 연구 | **부재** — ZnPCA+Niacinamide 단독 vs 병용 Sebumeter 비교 RCT 없음 |
| 시판 제품 근거 | The Ordinary Niacinamide 10% + Zinc 1% — 세계적 베스트셀러, 병용 효과 간접 입증 |

#### 실무 시사점
- 두 경로가 독립적 → **상가 효과 기대 합리적**
- 시너지는 아니지만, 각각 ~20-30%(Niacinamide) + ~45%(ZnPCA) 감소 시
  → 합산이 아닌 **독립 경로 상가: 1-(0.7×0.55) = 1-0.385 = ~62% 총 감소** 이론적 기대
- 이는 Ge et al. 임상(↓62%)과 일치하는 수치 (SA 포함이지만)

#### 판정: **지지** ✅
