# EXP-004: 세범 컨트롤 4-Active SF 샴푸 — 규제·시장·실무 종합

> 연구일: 2026-03-11
> 사이클: 3 (종합)
> 상위 프로젝트: 두피 세범 컨트롤 SF 샴푸
> 이전 실험: EXP-003 (4-Active 전달·호환성·경로 심화)

---

## 가설

> **H1:** 4-Active 조합(ZnPCA 0.3% + PO 0.5% + Gluconolactone 3% + Niacinamide 2%)이 EU, 중국 NMPA, ASEAN 규제에서 동시 적합하다
> **H2:** 2-Step 시스템(샴푸+두피 세럼)이 샴푸 단독 대비 유의미하게 우수한 세범 컨트롤 근거가 존재한다
> **H3:** SF 샴푸 시장에서 4-Active 멀티타겟 접근이 유효한 차별화 포지셔닝을 확보할 수 있다

#### Niacinamide 두피 적용 시사점

- **리브온에서 효과 극대화:** MW 122 Da, 수용성 → 충분한 접촉 시간 필요
- **린스오프에서는 보조적:** 표면 잔류 1-10%만 (EXP-003)
- **2-Step 전략에서 Niacinamide의 최적 위치:** 리브온 두피 세럼 (4%)

#### 실무 시사점

- 샴푸 단독은 세범 "세정"에는 효과적이나 "조절"은 한계
- 리브온 제품이 피지선 생산 조절에 유의미한 효과
- **2-Step 시스템의 메커니즘적 우위는 명확하나, 직접 비교 RCT는 부재**
- → 비교 임상 시험 설계 기회 (차별화 근거 확보 가능)

#### 판정: **부분 지지** ⚠️ (메커니즘적 근거 충분, 직접 임상 비교 부재)

```
Phase 1 (즉시): 한국 MFDS 신고 → 국내 출시
Phase 2 (3-6M): EU CPNP 등록 → 유럽 진출
Phase 3 (6-12M): NMPA 비안 → 중국 진출
Phase 4 (동시): ASEAN 회원국 개별 등록
```

### 3. 시장 차별화 5-Point 전략

1. **완전 SF (아미노산 계면활성제)** — SLES-free가 아닌 진정한 SF
2. **2-Step 시스템** — 샴푸+세럼 프로토콜로 결과 극대화
3. **임상 데이터** — Sebumeter 기반 자체 임상시험 (n≥30)
4. **K-발효 스토리** — 쌀 발효 여과물로 마이크로바이옴 내러티브
5. **아시아 시장 선점** — Pharmaceris 부재 지역 공략

### 4. Pharmaceris 벤치마크 발견 🔍

> **핵심 인사이트:** Pharmaceris H-Sebopurin이 이미 동일 4-Active를 사용 중

> → SF 베이스 + 2-Step + K-발효 + 임상 데이터로 **시스템 수준 차별화** 필수
