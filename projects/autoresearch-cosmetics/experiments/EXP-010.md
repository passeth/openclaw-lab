> **H1:** GABA-CERS3 경로 활성화를 목표로 한 당사 원료(Saccharomyces/Rice Ferment Filtrate) in vitro 실험 프로토콜 — 현재 문헌에서 HaCaT/CERS3/Filaggrin 직접 근거가 존재하는가, 그리고 실험 설계가 실현 가능한가

```
├─ 상가/시너지 가능 기전 (이론적):
│   ├─ Piroctone Olamine: 직접 Malassezia 억제 (항진균)
│   ├─ SRFF: 두피 장벽 강화 → 진균 침투 저항성↑ (간접 보호)
│   ├─ 두 기전이 보완적 → 이론적 시너지 ✅
│   └─ 그러나 항진균 활성 시너지 ≠ 장벽 강화 시너지 (혼동 주의)

├─ In Vitro 검증 방법론 (2024 신규 확인):
│   ├─ 브로스 미세희석법 (MIC 측정, M. furfur/M. restricta)
│   ├─ Dixon agar (지방산 보충 배지) 사용 필수
│   ├─ 2024 논문: Climbazole/Piroctone 조합 → 강력 억제 확인
│   └─ → 동일 방법으로 SRFF + Piroctone 조합 MIC 검증 가능
│
└─ 현재 샴푸 처방 맥락:
   ├─ SF 샴푸: SRFF 2% + Piroctone Olamine (현재 함량 미기록)
   ├─ 이미 조합 제형 → 조합의 in vitro 효능 데이터 필요
   └─ 단독 vs 조합 비교 실험 → 특허/클레임 강화 가능
```

#### 실무 설계

```

│   ├─ Control (배지만)
```
