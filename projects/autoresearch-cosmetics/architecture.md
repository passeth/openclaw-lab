## 도구 매핑

| 단계 | 도구 | 용도 | 필요 설정 |
|------|------|------|-----------|
| Step 1 | `web_search` (provider: grok) | 학술/기술 고급 검색 | xAI API 키 |
| Step 2 | Kimi 2.5 API 호출 | 검색 결과 분석 · 구조화 | Moonshot API 키 |
| Step 3 | InfraNodus API | 텍스트 네트워크 · 갭 분석 | InfraNodus API 키 |
| Step 4 | Kimi 2.5 API 호출 | 종합 평가 · 가설 도출 | (Step 2와 동일) |
| Step 5 | 파일 작성 | EXP 결과 문서화 | — |

## 사이클 전략

### 권장: 3사이클 / 주제

| 사이클 | 목적 | 깊이 |
|--------|------|------|
| **Cycle 1: 탐색** | 가설의 기본 검증, 핵심 문헌 수집 | 넓고 얕게 |
| **Cycle 2: 심화** | Cycle 1에서 발견된 갭/모순 파고들기 | 좁고 깊게 |
| **Cycle 3: 종합** | 실무 적용안 도출, 대안 비교 | 실용 중심 |

### 근거

- **2사이클 이하:** 표면적 조사에 그침, InfraNodus 갭 분석의 의미 희석
- **3사이클:** 탐색→심화→종합의 자연스러운 리듬. 각 사이클이 이전 결과를 기반으로 진화
- **4~5사이클:** 특별히 복잡한 주제(규제 + 화학 + 임상 교차)에만 필요
- **비용:** 사이클당 ~3회 외부 API 호출 (Grok 1 + Kimi 2 + InfraNodus 1)

### 종료 조건

자동 종료:
- 신뢰도 4/5 이상 도달
- 2연속 사이클에서 새로운 정보 없음
- 가설이 완전 기각되어 방향 전환 필요 (→ 메인에 보고)

수동 종료:
- 사용자 판단으로 충분한 근거 확보

## 오늘의 샴푸 프로젝트 연구 주제 후보

### 주제 1: Zinc PCA 안정화 전략 (EXP-001 심화)
- H1: Phytic Acid가 EDTA보다 Zn 전달효율을 보존하면서 안정화에 효과적이다
- H2: Glycinate:SLMI 비율 6:6이 8:4보다 Zn 안정성이 높다
- H3: 45°C 가속 안정성에서 pH 5.0이 5.5보다 Zinc PCA 침전이 적다

### 주제 2: SF 샴푸의 세범 컨트롤 액티브 최적 조합
- H1: Zinc PCA + Piroctone Olamine 병용이 단독보다 세범 감소율이 높다
- H2: Niacinamide 1%가 두피 환경에서 피지 조절에 유의미한 효과를 보인다
- H3: SA 없이도 PHA(Gluconolactone)가 두피 각질 관리에 효과적이다

### 주제 3: K-발효 성분의 두피 마이크로바이옴 근거
- H1: 쌀 발효 여과물이 두피 상재균 다양성을 유지/증가시킨다
- H2: 발효 추출물이 Piroctone Olamine의 항진균 효과를 보완한다
- H3: 발효 성분의 postbiotic 효과가 두피 장벽 강화에 기여한다
