## 2. 제품별 USP 정의

### 🟢 #1 지성두피 샴푸 — PURIFY

**원라인 USP:**
> **"저녁까지 가볍고 개운한 두피"**

| USP 요소 | 내용 | 근거 |
|----------|------|------|
| **농축 어성초** | 업계 평균 0.5~1% 대비 **2.0% 고농축** | PHARMA21/샤샤 벤치마크 초과 |
| **이중 유분 컨트롤** | 살리실산(각질) + 징크피리치온(피지) 동시 제어 | 표면 유분 + 모공 내 유분 이중 타격 |
| **어성초 캡슐 리추얼** | 캡슐을 직접 터뜨리는 세정 경험 | 유효성분 즉시 방출 + 감성적 사용 경험 |
| **쿨링 마무리** | 멘톨 0.3% → 세정 후 두피 시원함 | 지성두피 고객 최우선 니즈 = "개운함" |

**경쟁 대비 차별점:**
- 샤샤 어성초: 어성초만 의존 → EVAS는 어성초 + 살리실산 + 징크 **3중 시스템**
- 일반 쿨링 샴푸: 멘톨만 강함 → EVAS는 멘톨 + 실제 유분 제어 **체감+실효 동시**

---

### ⚪ #2 비듬케어 샴푸 — BALANCE

**원라인 USP:**
> **"비듬약 같은 효과, 화장품 같은 사용감"**

| USP 요소 | 내용 | 근거 |
|----------|------|------|
| **한방 × 사이언스 듀얼** | 용담근(한방 항진균) + 클림바졸+피록톤올아민(약리 항비듬) | 전통+현대 이중 접근 |
| **하얀색 = 청결의 시각화** | 순백 제형에 캡슐 포인트 | 비듬 고민 고객의 "깨끗함" 심리 직격 |
| **비듬 안 보이는 머리** | 항비듬 + 각질 연화(살리실산) + 두피 진정(알란토인) | 원인 제거 + 증상 완화 + 재발 방지 |
| **약용 느낌 탈피** | 스크럽 퓨어리프 향 → 깨끗하면서 세련된 향 | 니조랄/헤드앤숄더 "약냄새" 탈피 |

**경쟁 대비 차별점:**
- 니조랄: 효과 강하지만 사용감 최악 (뻣뻣, 약 냄새) → EVAS는 **약용 효과 + 데일리 샴푸 사용감**
- 헤드앤숄더: 대중적이지만 성분 단순 → EVAS는 **용담근 한방 스토리 + 이중 항비듬제**

---

### 🔴 #3 모근강화 샴푸 — STRENGTHEN

**원라인 USP:**
> **"빠지는 머리에서, 서는 머리로"**

| USP 요소 | 내용 | 근거 |
|----------|------|------|
| **측백엽 한방 포뮬러** | 동의보감 탈모 처방 핵심 약재 | 한방 탈모케어의 정통성 |
| **모근 4중 액티브** | 측백엽 + 카페인 + 비오틴 + 구리트리펩타이드 | 혈행(측백엽) + 모낭자극(카페인) + 성장(비오틴) + 성장인자(펩타이드) |
| **빨간 에너지** | 빨간색 제형 = 혈행, 활력, 에너지의 시각화 | 감성적 "이 샴푸가 일한다"는 인지 |
| **볼륨 체감** | 판테놀 0.5% + 케라틴 → 세정 후 모발 두께감 | 탈모 고민 고객의 숨은 니즈 = "숱이 많아 보이고 싶다" |

**경쟁 대비 차별점:**
- 닥터포헤어: 의약외품 느낌 강함, 감성 부족 → EVAS는 **한방 정통성 + 감각적 제형**
- 려(呂): 한방 라인이지만 측백엽 단독 주연은 없음 → EVAS는 **측백엽 전면 배치 + 펩타이드 현대 기술**
- TS 샴푸: 기능 위주 → EVAS는 **캡슐 경험 + 빨간 비주얼**로 사용 즐거움 추가

---

### 🟡 #4~5 손상모·영양케어 — REPAIR

**원라인 USP:**
> **"시카가 두피를 고치고, 케라틴이 모발을 채운다"**

| USP 요소 | 내용 | 근거 |
|----------|------|------|
| **CICA = 스킨케어 to 헤어케어** | 스킨케어에서 검증된 시카를 헤어로 확장 | 소비자 이미 "시카 = 리페어" 인식 학습됨 |
| **샴푸→트리트먼트 농도 상승 설계** | 핵심 성분 2~3배 증량 (CICA 1→1.5%, 케라틴 0.5→1%) | "쓸수록 채워진다" 스토리 |
| **저자극 세정** | SLES 최저 비율(7%) + SCI 최고 비율(4%) | 손상모에 추가 손상 최소화 |
| **노란색 = 영양의 시각화** | 강황+아르간+실크의 골든 컬러 → "영양 가득" | 시각적으로 "이건 영양 제품"이라는 즉각 인지 |
| **2-Step 시스템** | 유일하게 샴푸+트리트먼트 세트 | 라인 내 유일한 듀오 → "손상모는 2-Step이 필수" 메시지 |

**경쟁 대비 차별점:**
- 미쟝센 퍼펙트세럼: 실리콘 코팅 위주, 근본 보수 약함 → EVAS는 **시카+세라마이드 실제 리페어**
- 모로칸오일: 아르간 원료지만 초고가 → EVAS는 **아르간+시카+케라틴 복합 처방, 합리적 가격**

## 3.5 마케팅 액티브 — "소비자가 사는 이유"

> **원칙:** 서포트 액티브는 R&D가 선택, 마케팅 액티브는 소비자가 이해하고 공감하는 성분  
> **레퍼런스:** 르네휘테르(식물 테라피 스토리텔링) × 케라스타즈(프리미엄 원료 소싱 스토리)

### 🟢 지성두피 — 페퍼민트 + 대나무수

| 마케팅 액티브 | INCI | 스토리 | 레퍼런스 |
|-------------|------|--------|---------|
| **페퍼민트** | Mentha Piperita Oil | "알프스 페퍼민트의 쿨링 정화" — 이미 멘톨이 있지만 "에센셜 오일" 표기로 격상 | 르네휘테르 Forticea: 에센셜 오일 3종 블렌드 |
| **대나무수** | Bambusa Vulgaris Water | "제주 대나무의 미네랄 워터" — 미네랄 풍부, 두피 정화, 한국 소재 | 이니스프리 그린티 라인처럼 소재 원산지 스토리 |

**마케팅 카피:** *"어성초의 한방 정화력 + 페퍼민트 쿨링 + 대나무수 미네랄"*  
**소비자 언어:** "두피가 숨 쉬는 느낌"

### ⚪ 비듬케어 — 로즈마리 + 프로폴리스

| 마케팅 액티브 | INCI | 스토리 | 레퍼런스 |
|-------------|------|--------|---------|
| **로즈마리** | Rosmarinus Officinalis Extract | "지중해 로즈마리의 두피 정화" — 글로벌 헤어케어 트렌드 소재 #1 (TikTok 로즈마리워터 바이럴) | 르네휘테르 Melaleuca: 허브 기반 항비듬 |
| **프로폴리스** | Propolis Extract | "천연 항균 실드" — 벌이 만든 천연 방어막 스토리 | 케라스타즈 Genesis: 생강근 컨셉과 유사한 "자연의 방어 시스템" |

**마케팅 카피:** *"용담근 한방 + 로즈마리 허벌 + 프로폴리스 천연 항균"*  
**소비자 언어:** "비듬의 원인부터 잡는 자연 처방"

### 🔴 모근강화 — 인삼 + 생강

| 마케팅 액티브 | INCI | 스토리 | 레퍼런스 |
|-------------|------|--------|---------|
| **인삼** | Panax Ginseng Root Extract | "6년근 고려인삼의 모근 활력" — K-뷰티 최강 소재, 글로벌 인지도 최상 | 르네휘테르 Triphasic: Pfaffia(브라질 인삼) → 우리는 진짜 인삼 |
| **생강** | Zingiber Officinale Root Extract | "생강의 온열 혈행 촉진" — 모근에 피가 돈다는 직관적 이해 | 케라스타즈 Genesis: Ginger Root이 핵심 마케팅 액티브 |

**마케팅 카피:** *"측백엽 한방 + 고려인삼 활력 + 생강 온열 순환"*  
**소비자 언어:** "모근이 다시 깨어나는 한방 에너지"  
**보너스:** 르네휘테르는 Pfaffia(브라질 인삼)를 쓰는데, EVAS는 **진짜 고려인삼** → 원조의 힘

### 🟡 손상모 — 꿀 + 쌀추출물 (샴푸) / 동백 + 아이리스 (트리트먼트)

| 마케팅 액티브 | INCI | 스토리 | 레퍼런스 |
|-------------|------|--------|---------|
| **꿀** | Honey Extract | "영양의 상징" — 케라스타즈 Curl Manifesto의 마누카 허니와 같은 포지션 | 케라스타즈: Manuka Honey |

### 사용감 구현 방법

#### 🟢 지성두피 — "시원하고 가벼운 개운함"

| 사용감 | 구현 방법 |
|--------|----------|
| **쿨링감** | 멘톨 0.3% → 세정 시 즉각 쿨링 + 헹굼 후에도 잔여 청량감 |
| **빠른 거품** | SLES 10%(최고 비율) + 코카미드 MEA → 거품 빠르게 올라옴 |
| **가벼운 마무리** | PQ-10 최소량(0.3%) → 코팅감 없이 깨끗한 피니시 |
| **캡슐 터짐** | 녹색 캡슐이 거품 속에서 터지며 어성초향 확산 → "한방 세정 리추얼" |
| **"개운함" 시그널** | 멘톨 쿨링 + 어성초 허브향 → 감각적으로 "유분이 사라졌다" 인지 |

#### ⚪ 비듬케어 — "산뜻하고 청결한 편안함"

| 사용감 | 구현 방법 |
|--------|----------|
| **밀도 높은 거품** | SLES 9% + SLMI 3% → 촘촘하고 밀착 세정 (두피 커버리지 ↑) |
| **무자극 느낌** | 알란토인 0.2% + 판테놀 → 비듬으로 예민한 두피에 자극 없음 |
| **깨끗한 헹굼** | 잔여물 없는 포뮬러 → 비듬 고민 고객 "깨끗이 씻겼다" 확신 |
| **흰색 제형의 심리** | 순백 + 스크럽 퓨어리프 향 → 시각+후각 동시에 "청결" 시그널 |
| **캡슐 대비** | 흰색 바탕에 연갈색 캡슐 → 사용 전 시각적 흥미 + 전문성 인지 |

#### 🔴 모근강화 — "묵직한 활력, 두피가 깨어나는 느낌"

| 사용감 | 구현 방법 |
|--------|----------|
| **탄력 거품** | SCI 3% 비율 → 탄탄하고 찰진 거품 (가벼운 거품과 차별화) |
| **두피 자극감** | 카페인 0.3% → 미세한 온열감/자극감 → "모근에 뭔가 일어나고 있다" |
| **볼륨 드라이** | 판테놀 0.5% + 케라틴 → 드라이 후 모발이 서는 느낌, 숱 많아 보임 |
| **빨간색 임팩트** | 손에 짜는 순간 "강한 제품" 인지 → 플라시보 효과 증폭 |
| **캡슐 보색** | 빨간 바탕에 짙은녹색 캡슐 → 시각적 임팩트 최강 → 프리미엄 인지 |
| **머스크 잔향** | White Musk V2 → 세정 후 은은한 따뜻함 → "케어 받았다" 여운 |

#### 🟡 손상모·영양 — "실크처럼 부드러운 채움"

| 사용감 | 구현 방법 |
|--------|----------|
| **크리미 세정** | SLES 7%(최저) + SCI 4%(최고) → 거품이 부드럽고 크리미 |
| **슬립감** | 구아 하이드록시프로필 + PQ-10 0.5% → 세정 중 손가락이 미끄러지는 슬립 |
| **트리트먼트 밀착** | 세테아릴알코올+세틸알코올 → 두껍고 크리미한 텍스처, 모발에 밀착 |
| **부드러운 헹굼** | 아모디메치콘 → 손상 부위에 선택적 잔여 → 매끄러운 빗질 |
| **드라이 후 윤기** | 아르간+동백+실크 → 빛 반사 → "영양이 들어갔다" 시각적 확인 |
| **향 지속** | 트리트먼트 향료 0.4% → 12시간+ 은은한 잔향 |
| **캡슐 없는 이유** | 부드러움이 핵심 → 캡슐 알갱이가 오히려 텍스처 방해 |

### "강황이 연결하는 4가지 솔루션"

모든 제품에 강황이 공통으로 들어가 있다는 건:
1. **라인 통일성** — "같은 집안" 느낌
2. **강황 = 항산화 + 두피 진정** — 어떤 두피 고민이든 기본 케어
3. **골든 비주얼** — 캡슐 안에 강황 오일이 보이는 골드 포인트
4. **크로스 구매 유도** — "내 두피가 바뀌면 같은 라인 내 다른 제품으로"

### 캡슐의 마케팅 가치
