## 코스트다운 전략

| 전략 | 방법 | 절감 효과 |
|------|------|----------|
| **1. 계면활성제 믹스 변경** | SLMI 비율 ↓ → Cocamidopropyl Betaine ↑ + Sodium Cocoyl Glycinate 도입 | ★★★★★ (최대 절감) |
| **2. 고가 액티브 삭제/대체** | Copper Tripeptide → 삭제, Ceramide NP → Phytosphingosine 저가 대체 | ★★★★ |
| **3. 추출물 농도 최적화** | 어성초 2.0% → 0.5%, 기타 추출물 → 최소 유효 농도 | ★★★ |
| **4. 캡슐 사양 경량화** | 캡슐 0.5% → 0.3%, 쉘 재질 최적화 | ★★★ |
| **5. 점증/보조 원료 간소화** | PEG-150 DS → 저가 투명 점증 (NaCl + PEG-4000 혼합) | ★★ |

## CE에서도 절대 못 빼는 것 (사수 라인)

| 원료 | 이유 |
|------|------|
| **SLMI (최소 2.5~3%)** | Sulfate-Free 클레임 + 투명 제형의 핵심. 0%면 Betaine만으론 세정력 부족 |
| **캡슐 (#1~3)** | 바이어 필수 요청. 0.3%가 최소 (시각적 효과 유지) |
| **클림바졸 (#2)** | 비듬 효능의 핵심. 이걸 빼면 비듬케어 샴푸가 아님 |
| **멘톨 (#1)** | g당 1원 미만인데 사용감 기여도 최고. 가성비 왕 |
| **강황 추출물** | 전 라인 공통 DNA. 0.2%는 최소량 |
| **향료** | 0.2%가 최소 (향 지속 하한선) |

## CE 처방의 트레이드오프 (솔직한 평가)

| 희생되는 것 | 영향 | 대안 |
|-----------|------|------|
| 어성초 2% → 0.5% | 효능 체감 ↓ | 멘톨 쿨링으로 체감 보완 |
| Cu-Peptide 삭제 | 모근강화 과학적 근거 ↓ | 카페인+비오틴+나이아신 3종으로 커버 |
| Ceramide 삭제 | 손상모 보수력 ↓ | 케라틴 단독 + PQ-10 강화로 체감 유지 |
| 피록톤올아민 삭제 | 이중 항비듬 → 단일 | 클림바졸 0.5% 단독도 충분히 유효 |
| 추출물 전반 ↓ | 전성분 라벨 어필 ↓ | 순서상 앞에 오도록 배치 최적화 |
| Decyl Glucoside 삭제 | 마일드 세정 ↓ | Coco-Glucoside로 통합 (비슷한 효과) |

> 💡 **결론:** CE 처방은 "효능 70% + 사용감 85% + 원가 30%"  
> 소비자가 가장 체감하는 **거품, 향, 쿨링, 컨디셔닝**은 최대한 유지하고,  
> 체감 어려운 **고가 액티브(펩타이드, 세라마이드)**를 절감
