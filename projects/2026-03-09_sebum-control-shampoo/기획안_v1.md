## 1. 제형 처방 방향

### 세정 시스템: Option C (아미노산계) 추천

| 성분 | 배합 비율 | 역할 |
|------|-----------|------|
| Sodium Cocoyl Glycinate | 8~12% | 1차 세정, 약산성 적합, 아미노산계 마케팅 |
| Cocamidopropyl Betaine | 4~6% | 보조 세정 + 거품 보완 + 자극 완화 |
| Sodium Lauroyl Methyl Isethionate (SLMI) | 3~5% | 세정력 보강 + 크리미한 거품 |

**선정 근거:**
- 중국 프리미엄 샴푸 시장에서 **"氨基酸洗发水"(아미노산 샴푸)** 카테고리가 급성장 중
- Cocoyl Glycinate 주세정 → 아미노산계 클레임 확보
- SLMI 보조 → 아미노산계 단독 대비 세정력·거품감 보강
- Option A(Olefin Sulfonate)는 세정력은 강하나 "마일드" 포지셔닝과 충돌
- Option B도 가능하나, Option C가 중국 시장 스토리텔링에서 우위

### 제형 특성

| 항목 | 스펙 | 비고 |
|------|------|------|
| pH | 5.0 ~ 5.5 | 약산성, 두피 장벽 최적 |
| 점도 | 4,000 ~ 6,000 cps | 투명 젤 타입, 펌프 토출 적합 |
| 외관 | 투명 ~ 반투명 | 클린뷰티 이미지 |
| 색상 | 무색 ~ 연한 그린 | 자연·청결 이미지 |
| 향 | 시트러스 + 허벌 (유칼립투스/로즈마리) | 가벼운 톤, 청량감 연출 |
| 방부 | Phenoxyethanol (0.8%) + Sodium Benzoate (0.3%) | NMPA 허용 조합 |
| 용량 | 300mL / 500mL | 본품 기준 |

### 증점 시스템

- **Acrylates Copolymer** 또는 **PEG-150 Distearate + Cocamide MEA** 조합
- 투명 제형 유지 위해 PEG-150 Distearate 기반 추천
- 소금(NaCl) 증점은 아미노산계에서 효율 낮음 → 비이온 증점제 병행

## 2. 제품 기능 특성 5대 포인트

### Point 1. 🧴 12시간 세범 컨트롤 시스템

**컨셉:** 세안 후 12시간 이상 두피 유분 억제

| 성분 | 농도 | 메커니즘 |
|------|------|----------|
| Zinc PCA | 0.3% | 5α-reductase 억제 → 피지선 활성 억제 |
| Niacinamide | 1.0% | 피지 분비 속도 조절 + 두피 장벽 강화 |
| Piroctone Olamine | 0.5% | 말라세지아 억제 → 지루성 유분 감소 |

**기대 클레임:** 控油持久 / 12小时清爽 (12시간 프레시)  
**주의:** Zinc PCA는 음이온 계면활성제와 결합 시 침전 가능 → pH 5.0~5.5 유지 + 킬레이트제(EDTA-2Na 0.05%) 병행으로 안정성 확보

### Point 5. 🛡️ NMPA 규제 적합 설계

**컨셉:** 기획 단계부터 중국 규제 반영 → 등록 리스크 최소화

| 규제 항목 | 대응 방안 |
|-----------|-----------|
| 제품 분류 | **일반 화장품** 목표 (특수화장품 회피) |
| Salicylic Acid | ❌ 제외 → 특수화장품(去屑) 분류 리스크 |
| Piroctone Olamine | 0.5% 이하 → 일반화장품 범위 내 사용 |
| 방부제 | Phenoxyethanol + Sodium Benzoate (NMPA 허용) |
| 클레임 | 控油 (오일컨트롤), 蓬松 (볼륨) → 일반 클레임 가능 |
| 금지 클레임 | ~~去屑~~ (비듬), ~~防脱~~ (탈모) → 특수화장품 영역 |
| 성분 등록 | 전 성분 IECIC(已使用化妆品原料目录) 등재 확인 필요 |

**핵심 판단:**
- Salicylic Acid는 중국에서 "去屑"(비듬제거) 활성성분으로 분류될 가능성 높음 → **제외**
- 대신 Piroctone Olamine 0.5% + Zinc PCA 조합으로 기능 대체
- 控油/蓬松은 일반 화장품 클레임으로 가능하나, 去屑/防脱 표현은 절대 사용 불가
